## Why Variables Matter
Terraform variables remove hard-coded values and enable reusable, environment-agnostic infrastructure.

## Common Variable Types
- String (instance type)
- Boolean (enable/disable features)
- List (ports, subnets)

## Why Outputs Matter
Outputs expose important resource information after deployment and enable automation and module communication.

## SAA Perspective
Variables and outputs support predictable infrastructure changes, modular design, and safe scaling across environments.
